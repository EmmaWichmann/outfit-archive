# Transparent Clothing Image Workflow

Use this same process for every clothing piece:

1. Photograph one item against a plain background in bright, even light.
2. Remove the background with the iPhone Photos subject-lift tool, Canva
   Background Remover, Adobe Express, or another tool that supports transparency.
3. Export the finished image as a PNG. JPG files cannot contain transparency.
4. Open the PNG once before uploading it. A transparent background often appears
   as a gray-and-white checkerboard in an image editor.
5. Keep the image under 4 MB. A square image around 1200 by 1200 pixels is enough
   for this app.
6. In Outfit Archive, choose Add a piece and select that PNG.

Uploaded images are stored privately in that browser using `localStorage`.
They are not written into this folder or uploaded to a server. This also means
items saved on one device do not automatically appear on another device.
